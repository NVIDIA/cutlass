Tutorials
=========

Guided notebooks that walk through using the CUTLASS Operator API on concrete
examples. The notebooks are available in
`operators/examples/ <https://github.com/NVIDIA/cutlass/blob/main/operators/examples/>`__ on GitHub:

- `Core concepts and basic GEMM <https://github.com/NVIDIA/cutlass/blob/main/operators/examples/000_gemm.ipynb>`__
- `GEMM with fused epilogue <https://github.com/NVIDIA/cutlass/blob/main/operators/examples/001_gemm_with_fused_epilogue.ipynb>`__
- `Bringing your own kernel <https://github.com/NVIDIA/cutlass/blob/main/operators/examples/002_bring_your_own_kernel.ipynb>`__
- `Host latency best practices <https://github.com/NVIDIA/cutlass/blob/main/operators/examples/003_host_latency_best_practices.ipynb>`__
- `Fake tensors for compilation <https://github.com/NVIDIA/cutlass/blob/main/operators/examples/004_fake_tensors.ipynb>`__
- `Grouped GEMM with contiguous offset <https://github.com/NVIDIA/cutlass/blob/main/operators/examples/005_grouped_gemm_contiguous_offset.ipynb>`__
- `Block-scaled GEMM (MXFP8) <https://github.com/NVIDIA/cutlass/blob/main/operators/examples/006_block_scaled_gemm.ipynb>`__
